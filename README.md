# Excel Sales Dashboard 📊

This project is an interactive Excel dashboard built to analyse sales and profit data.

## 🔹 Features
- Pivot Tables for data analysis
- Charts for visualisation
- Slicers for interactivity
- KPI metrics for business insights

## 🔹 Insights
- Sales peak in November and December
- Regional performance varies across markets
- Category-wise analysis shows different trends

## 🔹 Tools Used
- Microsoft Excel
